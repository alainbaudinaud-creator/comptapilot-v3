1. Générer un token JWT

POST http://127.0.0.1:5000/ecritures/api/token

Body JSON :
{
    "username": "Alain BAUDINAUD",
    "password": "TON_MOT_DE_PASSE"
}

2. Appeler l’API protégée

GET http://127.0.0.1:5000/ecritures/api/v2/dashboard-financier

Header :
Authorization: Bearer TON_TOKEN_JWT

Endpoints disponibles

  • /ecritures/api/health
  • /ecritures/api/audit-comptable
  • /ecritures/api/dashboard-financier
  • /ecritures/api/token
  • /ecritures/api/v2/dashboard-financier
