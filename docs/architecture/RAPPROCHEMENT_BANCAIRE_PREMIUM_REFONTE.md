# Rapprochement bancaire premium — Refonte

Fonctionnalités :
- scoring montant / libellé / date,
- proposition automatique de rapprochement,
- statut PROPOSE ou A_RAPPROCHER,
- génération d’écriture banque en compte d’attente si absence d’écriture.

Objectif :
préparer un rapprochement bancaire automatique fiable et contrôlable par le cabinet.
