# API métier refonte — ComptaPilot V3

API isolée créée dans `app_refonte/routes/api_metier_demo.py`.

## Routes de démonstration
- `GET /api/refonte/health`
- `POST /api/refonte/immobilisation/amortissement`
- `POST /api/refonte/emprunt/tableau`
- `POST /api/refonte/tva/calcul`
- `GET /api/refonte/fec/controle-demo`

## Objectif
Tester les services métier premium avant intégration dans l’application principale.

## Sécurité
Cette API n’est pas encore branchée à la production.
Elle sert de banc d’essai contrôlé.
