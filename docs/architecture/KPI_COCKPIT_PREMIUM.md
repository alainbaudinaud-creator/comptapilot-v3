# KPI Cockpit Premium — ComptaPilot V3

Le moteur KPI isolé calcule :
- nombre de sociétés
- nombre d’écritures
- nombre de documents
- documents à traiter
- tâches urgentes
- alertes critiques
- équilibre débit/crédit
- score de production

Ce moteur est indépendant de la production et sera branché plus tard sur PostgreSQL.
