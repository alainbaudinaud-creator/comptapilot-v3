# Roadmap ComptaPilot V3 Premium

## Objectif
Créer une plateforme SaaS cabinet ultra professionnelle, moderne, rapide, commerciale, simple d’utilisation et conforme à la réglementation française.

## Modules prioritaires
1. Cockpit SaaS premium
2. Authentification fiable
3. Gestion cabinets / sociétés / collaborateurs
4. Production comptable
5. Plan comptable français
6. Saisie et import écritures
7. GED / OCR / IA documentaire
8. TVA / CA3 / contrôles fiscaux
9. FEC conforme DGFiP
10. Immobilisations et amortissements
11. Emprunts et tableaux d’amortissement
12. Rapprochement bancaire
13. Facturation électronique / PDP
14. Portail client
15. Workflow cabinet
16. Tableaux financiers temps réel
17. Journal d’audit et conformité RGPD

## Règle de développement
Aucune modification directe risquée sur la production.
Toute évolution doit passer par :
- branche de travail
- sauvegarde
- test syntaxe Python
- test import Flask
- test HTTP local
- commit
- puis seulement intégration
